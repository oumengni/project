销售数据分析

运用Python和Pandas对销售数据进行深度分析，挖掘销售趋势和顾客行为模式

2023年3月
数据分析
Python, Pandas, Matplotlib

项目概述

该项目旨在通过分析某电子商务平台三年的销售数据，识别销售趋势、顾客行为模式以及产品表现指标，为企业提供数据驱动的决策支持。通过深入的数据清洗、探索性分析和可视化，我们挖掘了对业务发展有价值的洞察。

项目使用Python生态系统中的Pandas进行数据处理，Matplotlib和Seaborn进行数据可视化，最终通过交互式仪表板展示分析结果，使决策者能够直观地理解复杂的数据关系。

数据探索

项目使用的数据集包含超过100万条销售记录，涵盖产品、价格、顾客、地区和时间等多个维度。在进行深入分析之前，我们首先对数据进行了全面的清洗和预处理：

数据清洗

处理缺失值、删除重复记录、标准化格式，确保数据质量。

特征工程

创建时间特征、计算客户生命周期值、分析购买频率等衍生指标。

探索性分析

通过描述性统计和可视化，初步理解数据分布和关系。

关键发现

销售趋势

  • 过去三年销售额年均增长率为18.5%，其中移动设备类产品贡献了最大增长。
  • 销售呈现明显的季节性模式，第四季度销售额平均比其他季度高出32%。
  • 周末销售转化率比工作日高15%，但平均订单金额较低。

客户行为

  • 高价值客户群（前20%）贡献了总销售额的65%，购买频率是普通客户的3倍。
  • 新客户获取成本持续上升，但客户留存率同样提高了8个百分点。
  • 移动端购物比例从2020年的35%上升到2023年的58%，表明渠道偏好的明显转变。

分析方法

本项目采用了多种数据分析技术和方法，包括：

时间序列分析

使用分解法识别销售数据中的趋势、季节性和残差成分，预测未来销售走势。

RFM客户细分

基于最近购买时间(Recency)、购买频率(Frequency)和消费金额(Monetary)对客户进行价值细分。

购物篮分析

通过关联规则挖掘产品之间的关联关系，优化产品组合和交叉销售策略。

数据分析的Python代码示例：

import pandas as pd
import numpy as np
import matplotlib.pyplot as plt
import seaborn as sns

# 加载数据
sales_data = pd.read_csv('sales_data.csv')

# 数据预处理
sales_data['order_date'] = pd.to_datetime(sales_data['order_date'])
sales_data['year'] = sales_data['order_date'].dt.year
sales_data['month'] = sales_data['order_date'].dt.month
sales_data['day'] = sales_data['order_date'].dt.day
sales_data['day_of_week'] = sales_data['order_date'].dt.dayofweek

# 按季度聚合销售数据
quarterly_sales = sales_data.groupby(pd.PeriodIndex(sales_data['order_date'], freq='Q'))['sales_amount'].sum().reset_index()
quarterly_sales['order_date'] = quarterly_sales['order_date'].astype(str)

# 可视化季度销售趋势
plt.figure(figsize=(12, 6))
sns.lineplot(x='order_date', y='sales_amount', data=quarterly_sales, marker='o', linewidth=2)
plt.title('季度销售趋势 (2020-2023)', fontsize=15)
plt.xlabel('季度')
plt.ylabel('销售额 (元)')
plt.xticks(rotation=45)
plt.grid(True, linestyle='--', alpha=0.7)
plt.tight_layout()
plt.savefig('sales_trend.jpg', dpi=300)
plt.show()

# RFM客户细分分析
rfm = sales_data.groupby('customer_id').agg({
    'order_date': lambda x: (pd.Timestamp('now') - x.max()).days,  # Recency
    'order_id': 'count',  # Frequency
    'sales_amount': 'sum'  # Monetary
}).rename(columns={
    'order_date': 'recency', 
    'order_id': 'frequency', 
    'sales_amount': 'monetary'
})

# 为RFM指标创建分位数
quantiles = rfm.quantile(q=[0.25, 0.5, 0.75])
rfm_segmentation = rfm.copy()

# 根据分位数对客户进行评分
for indicator, q_value in zip(['recency', 'frequency', 'monetary'], quantiles.columns):
    for i, q in enumerate(['25%', '50%', '75%']):
        if indicator == 'recency':
            rfm_segmentation[f'{indicator}_{i+1}'] = np.where(rfm[indicator] <= quantiles.loc[q, indicator], i+1, 0)
        else:
            rfm_segmentation[f'{indicator}_{i+1}'] = np.where(rfm[indicator] >= quantiles.loc[q, indicator], i+1, 0)

# 计算RFM评分
rfm_segmentation['r_score'] = rfm_segmentation['recency_1'] + rfm_segmentation['recency_2'] + rfm_segmentation['recency_3']
rfm_segmentation['f_score'] = rfm_segmentation['frequency_1'] + rfm_segmentation['frequency_2'] + rfm_segmentation['frequency_3']
rfm_segmentation['m_score'] = rfm_segmentation['monetary_1'] + rfm_segmentation['monetary_2'] + rfm_segmentation['monetary_3']
rfm_segmentation['rfm_score'] = rfm_segmentation['r_score'] + rfm_segmentation['f_score'] + rfm_segmentation['m_score']

# 客户细分
segment_labels = ['低价值', '中等价值', '高价值', 'VIP客户']
segment_thresholds = [0, 3, 6, 8, 9]
rfm_segmentation['customer_segment'] = pd.cut(
    rfm_segmentation['rfm_score'], 
    bins=segment_thresholds, 
    labels=segment_labels, 
    include_lowest=True
)

# 可视化客户细分
plt.figure(figsize=(10, 6))
segment_counts = rfm_segmentation['customer_segment'].value_counts().sort_index()
sns.barplot(x=segment_counts.index, y=segment_counts.values)
plt.title('客户细分分布', fontsize=15)
plt.xlabel('客户细分')
plt.ylabel('客户数量')
plt.tight_layout()
plt.savefig('customer_segments.jpg', dpi=300)
plt.show()

业务建议

基于数据分析结果，我们向企业提出以下战略建议：

  1. 聚焦高增长品类：增加智能家居和智能手机类产品的营销投入，这两个品类展现出最高的增长潜力和利润率。
  2. 优化季节性营销：根据销售季节性波动调整营销预算，在第四季度增加促销活动，在销售淡季开发特别促销计划。
  3. 客户细分策略：为VIP客户群体开发专属服务和产品，同时提高中等价值客户的转化率，扩大高价值客户基础。
  4. 移动端优化：鉴于移动端购物比例大幅增长，建议优先改进移动购物体验，加强APP功能和界面设计。
  5. 产品组合优化：基于购物篮分析结果，调整产品陈列和推荐系统，提高交叉销售和向上销售转化率。
